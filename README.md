# Horologium Advising Homepage

A simple static homepage for Horologium Advising.

## Files

- `index.html` — page structure
- `styles.css` — visual design and responsive layout
- `script.js` — mobile menu, reveal animations, and demo form behavior

## How to use on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.

GitHub will publish the site after the Pages build finishes.

## Note about the contact form

The form currently shows a demo success message. To collect real submissions, connect it to a service such as Formspree, Netlify Forms, Basin, or your own backend.
